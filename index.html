Proyectos

Nixbox | Network

Este proyecto inicio como un servidor pequeño con grandes expectativas, con el tiempo se fue haciendo mas grande y con mas modalidades pero como todo el proyecto cerro sus puertas en 2021 para segir adenlante con nuevos proyectos.

Drolph | Network

Este proyecto inicio como una network pequeña con grandes expectativas, con el tiempo se fue haciendo mas grande y con mas modalidades pero como todo el proyecto cerro sus puertas en el 2023 por temas personales y falta de dinero y tiempo.

Minebort | Network

Este proyecto inicio como una network donde estuve ayudando como developer y poco a poco fui subiendo de puesto hasta llegar a ser ceo de la network con falta de tiempo y jugadores el proyecto cerro sus puertas a finales de 2024.

Deleongui | Network

Este proyecto lo inicie con un amigo con arto presupuesto y experiencia al terminarlo de hacer se abrio como una network pequeña muy poco apoyo y al final cerro sus puertas en 2023 por los medias y falta de jugadores.

FrostMoond | Network

En este proyecto me contacto el dueño para solicitar mi ayuda como developer y ayudarlo a crear una network en mcserver poco a poco cerro sus puertas por falta de gente pero este año 2025 decidio reabrirla con nuevas modalidades y mejor configuracion y hosting actualmente soy ceo gracias ala ayuda dada y la satisfaccion del trabajo.

Clownmc | Network

En este tiempo el dueño del servidor me contacto para ayudarlo de forma externa osea sin acceso a consola para crear una solo modalidad con deceso de ser una network actualmente trabajo hay como developer y ayudo en lo que puedo devido al poco tiempo y apoyo que ya casi no tienen apesar de que lograron comprar hosting para mejor estabilidad.

Sunmc | Network

Me contacte con el dueño directamente para ayudarlo a crear su network con una modalidad y lobby unicos entre y gracias a mi esfuerzo logre ser owner del servidor actualmente ya no trabajo hay por temas personales.

Nautilus | Network

Este proyecto lo inicie con 2 amigos uno muy conocido por la gente y grandes creadores de contenido y el otro como developer con experiencia en diferentes lenguajes y configuraciones fue un proyecto muy avaro por la razon de tener nuestro propio hosting y hacer varias modalidades a gran escala actualmente cerro sus puertas este año 2025 junto con la oportunan de crecer y divertir a la gente.

MineFast | Network

Uno de mis amigos me contacto para ayudarlo como developer a crear su network con varias modalidades lastimosamente ya cerro se espera que vuelva a abrir con poco presupuesto y poca gente.

Rotium | Network

Me contacte con el dueño para ayudarlo en sus modalidades y que pudiera crecer mas de lo que ya era trabaje de manera externa y muy buena actualmente ya no trabajo hay por poca dedicacion pero es una de las network mas conocidas.
